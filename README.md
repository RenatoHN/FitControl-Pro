# FitControl Pro

Aplicación web local para registrar entrenamiento de fuerza, cardio, natación, recuperación, medidas corporales y alimentación.

## Historial precargado
Esta edición incluye el historial conocido de Renato hasta el 10 de julio de 2026:
- sesiones de fuerza de pecho, espalda, piernas, hombros y brazos;
- natación, caminadora y elíptica;
- sauna;
- peso y medidas corporales.

La información se carga una sola vez mediante una migración (`seedVersion: 2`) y después permanece en `localStorage`.

## Uso
Abra `index.html` en un navegador moderno. Los datos permanecen en el navegador. Use las opciones de exportación para crear copias de seguridad periódicas.
